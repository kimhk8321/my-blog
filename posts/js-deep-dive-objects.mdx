## 객체

객체는 **키(프로퍼티)와 값**의 집합입니다. 값으로 함수를 가지면 그 프로퍼티를 **메서드**라고 부릅니다.

```javascript
const person = {
  name: "형기",
  greet() {
    return `안녕하세요, ${this.name}입니다`;
  },
};

person.age = 30; // 동적으로 프로퍼티 추가
delete person.age; // 삭제
```

### 참조에 의한 전달

원시 타입은 **값**이 복사되지만, 객체는 **참조(주소)** 가 복사됩니다.

```javascript
const a = { x: 1 };
const b = a;
b.x = 99;
console.log(a.x); // 99 — 같은 객체를 가리킴
```

### 유용한 문법 (ES6+)

```javascript
const name = "형기";
const obj = {
  name, // 단축 프로퍼티
  [`key_${1}`]: "값", // 계산된 프로퍼티명
};

const merged = { ...obj, extra: true }; // 스프레드
const { name: n } = obj; // 구조 분해
```

## 함수

### 정의 방식

```javascript
function decl(a, b) { return a + b; } // 함수 선언문 (호이스팅됨)
const expr = function (a, b) { return a + b; }; // 함수 표현식
const arrow = (a, b) => a + b; // 화살표 함수
```

- **매개변수 기본값 / 나머지 매개변수**:

```javascript
function sum(first = 0, ...rest) {
  return rest.reduce((acc, n) => acc + n, first);
}
sum(1, 2, 3); // 6
```

- **일급 객체**: 함수를 변수에 담고, 인자로 넘기고, 반환할 수 있습니다. 고차 함수의 기반입니다.
- **IIFE(즉시 실행 함수)**: 정의와 동시에 실행해 스코프를 격리합니다. `(function () { … })();`

## 스코프와 실행 컨텍스트

- **스코프**: 식별자(변수)가 유효한 범위. 자바스크립트는 함수가 **선언된 위치**를 기준으로 상위 스코프를 결정하는 **렉시컬(정적) 스코프**를 따릅니다.
- **호이스팅**: 선언이 스코프 최상단으로 끌어올려지는 것처럼 동작. `var`는 `undefined`로 초기화되지만, `let`/`const`는 선언 전 접근 시 **TDZ(일시적 사각지대)** 로 에러가 납니다.

```javascript
console.log(a); // undefined (var는 호이스팅)
var a = 1;

console.log(b); // ReferenceError (TDZ)
let b = 1;
```

## 클로저 (Closure)

**함수가 선언될 당시의 렉시컬 환경을 기억**하여, 외부 함수가 종료된 뒤에도 그 변수에 접근할 수 있는 것을 클로저라고 합니다. 정보 은닉과 상태 유지에 쓰입니다.

```javascript
function createCounter() {
  let count = 0; // private 변수
  return {
    increment: () => ++count,
    get: () => count,
  };
}

const counter = createCounter();
counter.increment();
counter.increment();
counter.get(); // 2 — count는 외부에서 직접 접근 불가
```

## 프로토타입 (Prototype)

자바스크립트는 **프로토타입 기반** 객체지향 언어입니다. 모든 객체는 `[[Prototype]]`(`__proto__`)으로 다른 객체를 참조하며, 프로퍼티를 찾을 때 이 **프로토타입 체인**을 따라 올라갑니다.

```javascript
function Animal(name) {
  this.name = name;
}
Animal.prototype.speak = function () {
  return `${this.name} 소리`;
};

const dog = new Animal("멍멍이");
dog.speak(); // "멍멍이 소리" — 프로토타입에서 찾음
```

> `class` 문법도 내부적으로는 프로토타입 기반으로 동작하는 **문법적 설탕(syntactic sugar)** 입니다.

## this

`this`는 **함수가 어떻게 호출되었는지**에 따라 결정됩니다.

```javascript
function show() { return this; }

show(); // 일반 호출 → 전역객체(strict 모드는 undefined)
obj.method(); // 메서드 호출 → obj
new Show(); // 생성자 호출 → 새 인스턴스
show.call(ctx); // 명시적 바인딩 → ctx
```

- **화살표 함수**는 자신만의 `this`를 갖지 않고, **선언된 위치의 상위 `this`** 를 그대로 사용합니다(렉시컬 this). 콜백에서 `this`가 풀리는 문제를 피할 때 유용합니다.

```javascript
class Timer {
  seconds = 0;
  start() {
    // 화살표 함수라 this가 Timer 인스턴스를 유지
    setInterval(() => this.seconds++, 1000);
  }
}
```
