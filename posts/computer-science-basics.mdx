기술 면접과 기본기 점검을 위해 컴퓨터 과학의 핵심 개념을 영역별로 정리했습니다. 각 항목의 키워드를 빠르게 훑어보는 용도의 요약 노트입니다.

## 컴퓨터 구조 (Computer Architecture)

- **하드웨어**: 컴퓨터를 구성하는 기계적 장치 (CPU, RAM, HDD/SSD, 입력장치 등).
- **소프트웨어**: 하드웨어의 동작을 지시·제어하는 명령어 집합.
- **CPU**: 명령어를 해석·실행하는 중앙처리장치.
- **캐시 메모리**: CPU와 메모리 사이의 속도 차를 줄이는 고속 버퍼.
- **고정/부동 소수점**: 고정(정수+소수), 부동(지수+가수)으로 실수를 표현.
- **패리티 비트**: 전송 데이터에 1비트를 더해 **오류 검출**(검출만 가능).
- **해밍 코드**: 2ⁿ번째 자리에 패리티 비트를 두어 **오류 위치까지** 찾음.
- **ARM 프로세서**: Advanced RISC Machine. 감소된 명령 집합(RISC) 컴퓨팅.

## 자료구조 (Data Structure)

- **Array**: `map`, `filter`, `reduce`, `slice`, `splice`, `push/pop`, `shift/unshift` 등.
- **Linked List**: 연속된 메모리에 저장되지 않는 선형 구조.
- **스택(LIFO) / 큐(FIFO)**: JS 배열의 `push/pop`, `shift/unshift`로 구현.
- **Heap**: 이진 트리 기반. 우선순위 큐에 사용(JS 코테에선 보통 `sort`로 대체).
- **Tree**: value와 자식 노드 리스트로 구성.
- **이진 탐색 트리(BST)**: 노드별로 정렬 규칙을 추가한 트리.
- **Hash**: JS의 객체 `{}`. 평균 O(1) 조회.
- **Trie**: 문자열 검색을 빠르게 하는 구조. O(M) (M = 문자열 최대 길이).
- **B-Tree**: 이진 트리를 확장해 자식 수를 늘리고 균형을 맞춘 트리. DB·파일 시스템에서 널리 사용.

## 운영체제 (Operating System)

> 운영체제는 하드웨어를 관리하고 시스템 자원을 효율적으로 운용하며, 응용 프로그램과 하드웨어 사이의 인터페이스를 제공하는 시스템 소프트웨어다.

- **프로세스 & 스레드**: 실행 중인 프로그램 / 프로세스 내 실행 흐름 단위.
- **프로세스 주소 공간**: 코드, 데이터, 스택으로 구성.
- **인터럽트**: 실행 중 예기치 않은 상황 발생 시, 현재 작업을 중단하고 우선 처리를 CPU에 알림.
- **System call**: `fork`, `exec`(새 프로세스 생성), `wait`(자식 종료 대기).
- **PCB & Context Switching**: 프로세스 정보 블록과 문맥 교환.
- **IPC(Inter Process Communication)**: 독립적인 프로세스 간 통신.
- **CPU 스케줄링**: CPU 사용 순서 결정.
- **데드락**: 둘 이상의 프로세스가 자원을 얻지 못해 다음 처리를 못 하는 상태.
- **Race Condition**: 공유 자원에 동시 접근해 결과가 달라질 수 있는 상태.
- **세마포어 / 뮤텍스**: 공유 자원 접근 제어. 세마포어는 카운팅 기반, 뮤텍스는 상호 배제(한 번에 하나).
- **메모리 관리 / 파일 시스템**: 주기억장치의 동적 분할, 파일의 체계적 관리.

## 데이터베이스 (Database)

- **Key**: 튜플을 구분하는 기준 속성.
- **Join**: 둘 이상의 테이블을 연결해 데이터를 검색.
- **SQL Injection**: 조작된 SQL이 그대로 실행되는 공격 기법.
- **SQL vs NoSQL**: 정형 스키마(MySQL) vs 유연한 스키마(MongoDB).
- **Anomaly(이상 현상)**: 잘못된 테이블 설계로 발생 → 정규화가 필요한 이유.
- **Index**: 컬럼을 색인화해 검색 속도를 높이는 기술.
- **정규화**: 중복을 줄이고 무결성을 높이는 설계.
- **트랜잭션**: DB 상태를 바꾸는 작업의 단위(ACID).
- **Redis**: RAM에 저장하는 인메모리 Key-Value 스토어(빠르지만 휘발성 → 백업 필요).
- **저장 프로시저**: 일련의 쿼리를 함수처럼 실행하는 쿼리 집합.
