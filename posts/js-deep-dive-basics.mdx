『모던 자바스크립트 Deep Dive』를 읽으며 정리한 기초 문법 노트입니다. 프로그래밍의 개념부터 타입 변환까지 1~9장의 핵심을 다룹니다.

## 1. 프로그래밍

- **프로그래밍**이란 컴퓨터에게 실행을 요구하는 일종의 커뮤니케이션입니다. 0과 1밖에 모르는 기계가 실행할 수 있을 만큼 정확하고 상세하게 요구사항을 설명하는 작업이며, 그 결과물이 코드입니다.
- 기계어는 사람의 언어와 체계가 달라 직접 명령하기 어렵기 때문에, **컴파일러** 또는 **인터프리터**를 사용합니다.
  - **컴파일러**: 고급 언어를 실행 프로그램(저급 언어)으로 한 번에 변환 (예: C, Java)
  - **인터프리터**: 소스 코드를 한 줄씩 바로 실행 (예: Python, JavaScript)
- 프로그래밍 언어는 "구문(syntax)"과 "의미(semantics)"의 조합으로 표현됩니다.

## 2. 자바스크립트란?

- **탄생**: 1995년, 웹페이지의 보조 기능을 위해 브라우저에서 동작하는 경량 언어로 도입(모카 → 라이브스크립트 → 자바스크립트).
- **표준화**: 파생 버전 JScript가 나오며 **크로스 브라우징** 이슈가 발생 → 표준화 필요성 대두 → **ECMAScript** 탄생. (크로스 브라우징: 서로 다른 브라우저에서 웹을 비슷하게 보여주는 작업)
- **성장의 역사**:
  - **Ajax**: 자바스크립트로 서버와 **비동기** 데이터를 교환하는 통신 기능. 페이지 전체를 새로고침하지 않고 필요한 데이터만 주고받습니다. (오늘날에는 `XMLHttpRequest`보다 `fetch`나 `axios`를 주로 사용합니다 — `fetch`는 브라우저 내장 Promise 기반 API, `axios`는 인터셉터·자동 JSON 변환 등을 제공하는 외부 라이브러리)
  - **jQuery**: DOM 제어를 쉽게 하고 크로스 브라우징 이슈를 완화.
  - **V8 엔진**: 자바스크립트를 바이트코드로 컴파일·실행해 속도를 크게 끌어올림. 서버에서 하던 로직이 클라이언트로 이동.
  - **Node.js**: 자바스크립트 엔진을 브라우저에서 독립시킨 런타임. 서버 사이드 개발에 주로 쓰이며 파일 시스템·HTTP 등 내장 API를 제공.
  - **SPA 프레임워크**: Angular, React, Vue, Svelte 등 등장. (SPA: 하나의 페이지에서 화면을 동적으로 바꾸는 방식, CBD: 컴포넌트 기반 개발)
- **특징**: 인터프리터 언어이지만 모던 엔진은 인터프리터와 컴파일러의 장점을 결합했습니다. 명령형·함수형·**프로토타입 기반 객체지향**을 지원하는 멀티 패러다임 언어입니다.

## 3. 개발 환경과 실행

- **Node.js**: 자바스크립트를 브라우저 밖에서 실행하는 환경. Babel, Webpack, ESLint 등 도구를 쓰려면 필요합니다.
- **npm**: 자바스크립트 패키지 매니저. 모듈 저장소 역할과 설치·관리 CLI를 제공합니다.

## 4. 변수

- **변수**: 메모리에 저장된 값을 재사용하기 위한 것이며, 변수 이름을 **식별자**라고 합니다.
- 선언 키워드: `var`(함수 레벨 스코프, 호이스팅 있음), `let`(재할당 가능), `const`(재할당 불가).
- 모든 식별자는 **실행 컨텍스트**에 등록됩니다. 실행 컨텍스트는 엔진이 코드를 평가·실행하기 위한 환경으로, 식별자와 스코프를 관리합니다.
- 어떤 식별자와도 연결되지 않은 값은 **가비지 컬렉터**가 자동으로 메모리에서 해제합니다.
- 식별자는 문자·숫자·언더스코어·달러 기호를 쓸 수 있으나 **숫자로 시작할 수 없습니다.**
- 네이밍 컨벤션: 카멜 케이스, 스네이크 케이스, 파스칼 케이스.

## 5. 표현식과 문

- **값**: 식이 평가되어 생성된 결과. `10 + 20`은 평가되어 값 `30`을 생성합니다.
- **리터럴**: 사람이 이해할 수 있는 기호로 값을 생성하는 표기법 (예: 숫자, 문자열).
- **표현식**: 값으로 평가될 수 있는 문. `var x;`(선언문)는 평가될 수 없으므로 표현식이 아니지만, `var x = 3;`은 표현식입니다.
- **문(statement)**: 프로그램을 구성하는 기본·최소 실행 단위. 변수 선언문, 할당문, 조건문, 반복문 등.
- **토큰**: 문법적으로 더 이상 나눌 수 없는 코드의 기본 요소. `var`, `person`, `=`, `"형기"`는 모두 토큰입니다.

## 6. 데이터 타입

- **원시 타입**: 숫자, 문자열, 불리언, `undefined`, `null`, 심벌(유일무이한 값).
- **객체 타입**: 객체, 함수, 배열.
- **숫자 타입**: 하나만 존재합니다(`int`/`float` 구분 없음).
- **문자열 타입**: 변경 불가능한 값. 값을 "바꾸면" 기존 값은 두고 새 메모리를 할당하며, 변수가 가리키는 값이 바뀌는 것입니다.
- **템플릿 리터럴**(ES6): `` `hello ${name}` ``
- 컴파일러·인터프리터는 **심벌 테이블**로 식별자를 키 삼아 메모리 주소·타입·스코프를 관리합니다.
- **정적 타입**(TypeScript): 타입을 사전에 정의 / **동적 타입**(JavaScript): 사전에 정의하지 않음.

## 7. 연산자

- 종류: 산술, 할당, 비교, 논리, 타입, 지수 연산자.
- **암묵적 타입 변환**(타입 강제 변환):

```javascript
"1" + 2; // "12"   (문자열 연결)
1 + true; // 2     (true → 1)
1 + false; // 1    (false → 0)
```

- 동등(`==`) / 일치(`===`) 비교. 예외로 **`NaN`은 자기 자신과도 일치하지 않습니다.**
- 지수 연산자: 기존 `Math.pow` 대신 `**` 사용.
- 그 외: `?.`(옵셔널 체이닝), `??`(null 병합), `delete`(프로퍼티 삭제), `new`(인스턴스 생성), `instanceof`(인스턴스 판별).

## 8. 제어문

- **블록문**: 0개 이상의 문을 `{}`로 묶은 것. 자체 종결성이 있어 `;`가 필요 없습니다.
- **조건문**: `if…else`, `switch`. `switch`에서 `break` 없이 다음 case로 넘어가는 것을 **폴스루(fall-through)** 라고 합니다.
- **반복문**: `for`, `while`, `do…while`.

```javascript
let count = 0;
do {
  console.log(count); // 0 1 2
  count++;
} while (count < 3);
```

## 9. 타입 변환과 단축 평가

- **문자열로 변환**: `+` 연산자는 피연산자 중 하나라도 문자열이면 문자열 연결로 동작.
- **숫자로 변환**: `-`, `*`, `/` 연산자는 숫자가 아닌 피연산자를 숫자로 암묵 변환.
- **불리언으로 변환**: Truthy/Falsy 값을 `true`/`false`로 암묵 변환.
