안녕하세요. 프론트엔드 개발자로서 배우고 경험한 것들을 기록하기 위해 이 블로그를 시작합니다.

## 왜 블로그를 시작하나요

개발을 하면서 배운 것들은 시간이 지나면 흐릿해지기 마련입니다. 글로 정리해 두면 다음과 같은 점이 좋다고 생각합니다.

- **학습의 정리**: 머릿속의 지식을 글로 풀어내면서 이해가 깊어집니다.
- **기록과 회고**: 과거의 고민과 해결 과정을 다시 들여다볼 수 있습니다.
- **공유**: 비슷한 문제를 겪는 다른 개발자에게 도움이 될 수 있습니다.

## 어떤 글을 쓸 예정인가요

주로 프론트엔드 생태계를 중심으로 다룰 예정입니다.

1. React / Next.js 와 같은 프레임워크 학습 기록
2. 타입스크립트와 웹 표준에 대한 정리
3. 성능 최적화, 접근성, 사용자 경험에 대한 고민
4. 사이드 프로젝트와 회고

코드 예시도 함께 다룹니다.

```ts
type Greeting = `Hello, ${string}!`;

function greet(name: string): Greeting {
  return `Hello, ${name}!`;
}

console.log(greet("world"));
```

## 앞으로의 방향

꾸준함이 가장 어렵지만 가장 중요하다고 생각합니다. 완벽한 글보다는 **꾸준한 기록**을 목표로, 하나씩 채워 나가겠습니다.

읽어 주셔서 감사합니다. 🙌
